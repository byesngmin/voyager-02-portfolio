# 애니메이션 연출 용어 사전

연출 명령 시 이 문서의 용어를 그대로 사용할 수 있다.
예: "이 요소에 `스태거 인` 적용해줘", "스프링의 `댐핑`을 낮춰줘"

---

## 1. 진입/퇴장 트랜지션 (Entrance / Exit)

| 영어 용어 | 한국어 명칭 | 설명 | 명령 예시 |
|---|---|---|---|
| Fade in / out | 페이드 인/아웃 | 불투명도가 서서히 변하며 등장/퇴장 | "모달에 페이드 인 효과 넣어줘" |
| Slide in / out | 슬라이드 인/아웃 | 방향에서 미끄러지듯 등장/퇴장 | "사이드바가 왼쪽에서 슬라이드 인 되게 해줘" |
| Zoom in / out | 줌 인/아웃 | 크기가 변하면서 등장/퇴장 | "팝업이 화면 중앙에서 줌 인으로 나타나게 해줘" |
| Reveal | 리빌 | 마스크가 벗겨지며 내용이 드러남 | "텍스트가 위로 올라오면서 리빌 되도록 해줘" |
| Wipe | 와이프 | 한쪽 방향으로 닦아내듯 전환 | "화면 전환 시 좌→우 와이프 효과를 넣어줘" |
| Iris | 아이리스 | 원형으로 조여지거나 넓어지며 전환 | "다음 장면으로 넘어갈 때 아이리스 아웃 되게 해줘" |

---

## 2. 이징/타이밍 함수 (Easing / Timing Curve)

| 영어 용어 | 한국어 명칭 | 설명 | 명령 예시 |
|---|---|---|---|
| Linear | 리니어 | 일정한 등속도 움직임 | "진행 바를 리니어 타이밍으로 채워줘" |
| Ease In | 이지 인 (가속) | 천천히 시작해서 점점 빨라짐 | "로켓이 발사될 때 이지 인 함수를 적용해줘" |
| Ease Out | 이지 아웃 (감속) | 빠르게 시작해서 끝에 서서히 멈춤 | "메뉴가 나타날 때 이지 아웃으로 감속하며 멈추게 해줘" |
| Ease In-Out | 이지 인-아웃 (가감속) | 천천히 → 빨라졌다가 → 다시 천천히 | "요소가 왕복할 때 이지 인-아웃을 써줘" |
| Cubic Bezier | 큐빅 베지에 | 제어점으로 속도를 정밀하게 튜닝 | "통통 튀는 느낌을 위해 커스텀 큐빅 베지에를 적용해줘" |
| Step | 스텝 | 중간 프레임 없이 단계별로 뚝뚝 전환 | "도트 로딩 아이콘이 틱틱 돌도록 스텝 이징을 써줘" |

## 3. 변형 종류 (Transform Types)

| 영어 용어 | 한국어 명칭 | 설명 | 명령 예시 |
|---|---|---|---|
| Translate | 이동 (트랜슬레이트) | X/Y/Z 축으로 위치 이동 | "호버 시 위로 10px 트랜슬레이트 시켜줘" |
| Scale | 크기 조절 (스케일) | 비율로 확대/축소 | "호버 시 이미지를 1.1배 스케일 키워줘" |
| Rotate | 회전 (로테이트) | 중심점 기준으로 각도 회전 | "톱니바퀴 아이콘이 무한 로테이트 되게 해줘" |
| Skew | 기울임 (스큐) | 모서리 각도를 비틀어 기울이기 | "텍스트를 스큐로 기울여 속도감을 줘" |
| Flip | 뒤집기 (플립) | 3D 공간에서 앞뒤로 완전히 뒤집기 | "카드 앞뒷면이 바뀌도록 3D 플립 애니메이션을 해줘" |
| Perspective | 원근감 (퍼스펙티브) | 3D 변형 시 카메라 거리감 부여 | "원근감을 주어서 깊이 있게 회전시켜줘" |

## 4. 반복/루프 패턴 (Loop / Repeat)

| 영어 용어 | 한국어 명칭 | 설명 | 명령 예시 |
|---|---|---|---|
| Infinite Loop | 무한 루프 | 끝없이 동일한 애니메이션 반복 | "배경 구름이 무한 루프로 흘러가게 해줘" |
| Ping-pong / Alternate | 핑퐁 / 왕복 | 정방향 → 역방향 교대 반복 | "아이콘이 위아래로 핑퐁 루프 되도록 해줘" |
| Yoyo | 요요 | 끝점 → 시작점으로 되감기듯 반복 | "크기가 커졌다 작아지는 요요 효과를 반복해줘" |
| Counted Repeat | 횟수 지정 반복 | 정해진 횟수만큼만 반복 후 정지 | "경고 알림이 딱 3번만 흔들리게 해줘" |
| Seamless Loop | 심리스 루프 | 시작/끝이 완벽히 이어져 끊김 없는 루프 | "배경 패턴이 심리스 루프로 흐르게 해줘" |
| Hold / Freeze | 홀드 / 정지 | 마지막 프레임 상태로 고정 | "3번 반짝인 후 밝은 상태로 홀드 시켜줘" |

## 5. 시퀀스/오케스트레이션 (Stagger / Sequence)

| 영어 용어 | 한국어 명칭 | 설명 | 명령 예시 |
|---|---|---|---|
| Stagger | 순차 지연 (스태거) | 여러 요소가 시차를 두고 도미노처럼 등장 | "목록 아이템들이 0.1초 간격으로 스태거 인 되게 해줘" |
| Sequence | 시퀀스 | 하나 완료 후 다음 애니메이션이 이어지는 흐름 | "로고가 나타난 후 텍스트가 뜨는 시퀀스로 해줘" |
| Delay | 지연 (딜레이) | 시작 전 대기 시간 | "페이지 로드 후 1초 딜레이 주고 팝업을 띄워줘" |
| Overlap | 겹침 (오버랩) | 이전 동작이 끝나기 전에 다음 동작 시작 | "아이템 등장 시 오버랩 되어서 속도감 있게 해줘" |
| Keyframes | 키프레임 | 시점(%)별 상태 지정으로 복잡한 애니메이션 구성 | "여러 단계에 걸쳐 색상·크기가 변하도록 키프레임을 짜줘" |
| Timeline | 타임라인 | 여러 요소의 시점과 순서를 통합 제어 | "세 요소의 등장 시점을 타임라인으로 정밀하게 맞춰줘" |

## 6. 강조 효과 (Emphasis / Attention Seekers)

| 영어 용어 | 한국어 명칭 | 설명 | 명령 예시 |
|---|---|---|---|
| Pulse | 맥박 (펄스) | 크기/불투명도가 커졌다 작아지며 박동 | "결제 버튼에 펄스 효과로 시선을 유도해줘" |
| Shake / Quake | 흔들림 (셰이크) | 좌우 혹은 상하로 빠르게 흔들리는 효과 | "비밀번호가 틀렸을 때 입력창에 셰이크를 줘" |
| Flash / Blink | 깜빡임 (플래시) | 순간 밝아지거나 보였다 안 보였다 반복 | "알림 뱃지가 눈에 띄게 플래시 되게 해줘" |
| Bounce | 바운스 (강조) | 제자리에서 통통 튀어오르며 이목을 끌기 | "에러 메시지가 나타날 때 바운스 효과를 넣어줘" |
| Jello | 젤로 | 젤리처럼 양옆으로 출렁이며 형태 왜곡 | "클릭 시 아이콘이 젤로처럼 쫀득하게 출렁이게 해줘" |
| Wobble | 워블 | 무게 중심이 흔들리며 뒤뚱거리는 효과 | "삭제 대기 항목이 워블되면서 흔들리게 해줘" |

## 7. 물리 기반 / 탄성 (Spring / Physics)

| 영어 용어 | 한국어 명칭 | 설명 | 명령 예시 |
|---|---|---|---|
| Spring | 탄성 (스프링) | 목표값을 넘었다 돌아오며 정착하는 용수철 효과 | "모달이 열릴 때 텐션 있는 스프링 애니메이션을 써줘" |
| Gravity / Bounce | 중력 / 바운스 | 중력에 의해 떨어져 바닥에 튕기는 물리 효과 | "공 아이콘이 위에서 떨어져 물리적으로 바운스 되게 해줘" |
| Damping | 감쇠력 (댐핑) | 스프링 진동을 멈추게 하는 마찰력 | "댐핑을 높여서 덜 출렁거리게 잡아줘" |
| Mass | 질량 (매스) | 요소의 가볍고 무거운 정도 제어 | "매스를 무겁게 설정해서 묵직하게 움직이게 해줘" |
| Stiffness / Tension | 강성 / 장력 | 스프링 복원력의 세기 | "스티프니스를 높여서 빠르고 쫀쫀하게 반응하게 해줘" |
| Velocity | 관성 속도 (벨로시티) | 드래그/스크롤을 놓았을 때 유지되는 초기 속도 | "스와이프 후 관성이 자연스럽게 이어지게 해줘" |

## 8. 스크롤 연동 (Scroll-linked)

| 영어 용어 | 한국어 명칭 | 설명 | 명령 예시 |
|---|---|---|---|
| Parallax | 시차 (패럴랙스) | 배경/전경 이동 속도 차이로 입체감 | "스크롤 시 배경이 더 천천히 움직이는 패럴랙스를 넣어줘" |
| Scroll Trigger | 스크롤 트리거 | 요소가 뷰포트에 도달 시 애니메이션 시작 | "섹션이 화면 중심에 오면 애니메이션을 재생해줘" |
| Scroll Progress | 스크롤 진행도 | 스크롤 위치 비율에 동기화해 재생 | "상단 진행 바가 스크롤 진행도에 맞춰 채워지게 해줘" |
| Sticky | 고정 (스티키) | 특정 구간 동안 요소가 화면에 고정됨 | "스크롤 내려도 이미지가 상단에 스티키 되게 해줘" |
| Reveal on Scroll | 스크롤 리빌 | 뷰포트 진입 시 요소가 나타나는 효과 | "스크롤 내리면 텍스트가 아래에서 위로 나타나게 해줘" |
| Scrubbing | 스크러빙 | 스크롤 위치로 타임라인 앞뒤 재생 | "스크롤에 맞춰 3D 모델이 턴테이블처럼 돌아가게 해줘" |

## 9. 상태 전환 (State Transition)

| 영어 용어 | 한국어 명칭 | 설명 | 명령 예시 |
|---|---|---|---|
| Morph | 형태 변형 (모프) | 하나의 모양이 다른 모양으로 매끄럽게 변환 | "재생 버튼을 클릭하면 정지 아이콘으로 모프 해줘" |
| Crossfade | 교차 디졸브 | 하나는 페이드 아웃, 다른 하나는 페이드 인으로 교차 | "이미지 슬라이더를 크로스페이드로 전환해줘" |
| Shared Element | 공유 요소 전환 | 목록 아이템이 상세 페이지 헤더로 이어지며 연결 | "상품 클릭 시 썸네일이 커지며 상세 페이지로 이어지게 해줘" |
| Color Tween | 색상 전환 | 색상이 부드럽게 점진적으로 변함 | "다크 모드 전환 시 배경이 검정으로 컬러 트윈 되게 해줘" |
| Expand / Collapse | 확장 / 축소 | 숨겨진 영역이 부드럽게 열리거나 닫힘 | "상세 탭 클릭 시 영역이 아래로 익스팬드 되게 해줘" |
| FLIP | FLIP 애니메이션 | 위치/크기 변화를 역연산으로 부드럽게 연결하는 기법 | "리스트 순서가 바뀔 때 FLIP으로 자연스럽게 이동시켜줘" |

## 10. 파티클 / VFX (Particle / VFX)

| 영어 용어 | 한국어 명칭 | 설명 | 명령 예시 |
|---|---|---|---|
| Confetti | 콘페티 (꽃가루) | 축하 이벤트 시 화면에 흩날리는 파티클 | "폼 제출 성공 시 콘페티 터뜨리게 해줘" |
| Ripple | 물결 (리플) | 클릭 위치에서 원형 파동이 퍼져나가는 피드백 | "버튼 클릭 시 리플 이펙트를 넣어줘" |
| Glow / Aura | 발광 (글로우) | 요소 주변에 빛이 번지듯 일렁이는 효과 | "희귀 아이템 아이콘에 글로우 효과가 맥동하게 해줘" |
| Sparkle | 반짝임 (스파클) | 작은 빛 입자들이 순간 생성·소멸하는 효과 | "새 기능 아이콘에 스파클 이펙트를 추가해줘" |
| Trail / Ghost | 잔상 (트레일) | 빠르게 이동하는 객체 뒤에 남는 궤적 | "마우스 커서를 따라다니는 트레일 효과를 만들어줘" |
| Glitch | 글리치 (디지털 깨짐) | 화면/텍스트가 지지직거리며 깨지는 사이버펑크 효과 | "로딩 타이틀에 글리치 효과를 짧게 넣어줘" |
